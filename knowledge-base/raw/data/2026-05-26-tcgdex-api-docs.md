## Excerpts

Overview excerpts:

"The Multilingual Pokemon TCG API"

"Every bit of information from your Cards freely available and open source!"

The homepage lists:

- "10 languages and counting"
- a JSON REST API for cards, sets, series, and other fields
- a GraphQL API
- SDKs for Java, Kotlin, PHP, JavaScript, TypeScript, and Python

FAQ excerpts:

"No. The TCGdex API is free to use and requires no API key."

"There are no published hard rate limits, but please be considerate. For bulk data needs, cache responses locally rather than fetching the same data repeatedly."

"The API supports over 10 languages including English, French, Spanish, German, Italian, Portuguese (Brazilian), Japanese, Chinese (Traditional), Indonesian, and Thai."

Pokémon TCG Live integration excerpts:

"The TCGdx API provides access to all Pokémon cards available in Pokémon TCG Live."

"Pokémon TCG Live replaced Pokémon TCG Online (PTCGO) in 2023 as the official digital TCG platform."

"From an API perspective, both platforms use identical card data since they’re based on the physical TCG."

The integration page emphasizes full physical-card-pool coverage, multilingual card retrieval, and search/filter support for TCG Live applications.

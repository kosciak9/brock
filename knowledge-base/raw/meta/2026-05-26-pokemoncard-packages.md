Page heading: "Public Packages"

The page presents user-created card groups/packages with fields such as:

- package name
- description
- author
- card count
- subscriber count

Examples include small card modules like "Cynthia's Garchomp ex Package" as well as full 60-card lists and tiny 1–4 card bundles.

This suggests a modular way of thinking about deck construction and sharing: not only whole decks, but reusable sub-packages.

## Summary

Four reusable learning patterns stand out for a Pokémon practice tool:

1. retrieval over rereading
2. spacing over cramming
3. interleaving over monotony
4. reflection plus action items over vague review

## Retrieval practice

The Learning Scientists argue that recalling from memory is more powerful than re-reading. For a TCG tool, that maps naturally to:

- turn quizzes
- matchup flashcards
- opening-hand keeps/mulls from memory
- "write your line before seeing feedback"

## Spaced practice

The same source set argues for repeated short sessions over one long cram. For a TCG tool, that suggests:

- repeat weak matchup drills across days
- re-surface old mistakes after time passes
- rotate a weekly focus rather than grinding one concept once

## Interleaving

Interleaving argues against over-blocked practice. For a TCG tool, that suggests:

- mix sequencing, prize mapping, and matchup drills
- rotate between archetypes and board states
- avoid letting users spam only one familiar scenario

## Retrospectives and action items

Atlassian's retrospective guidance adds an important behavior layer:

- gather feedback
- look for patterns and trends
- create action items

That maps well to post-game review. A strong practice tool should not end at "here’s what happened"; it should end at "here’s the next thing to practice."

## Synthesis

These sources collectively push toward a practice product that is:

- active, not passive
- repeated, not one-and-done
- mixed, not monotonous
- action-oriented, not just analytical
